This project is an interactive Swiggy Dashboard created using Microsoft Excel to analyze food delivery data and visualize important business insights. The dashboard helps in understanding customer behavior, restaurant performance, and order trends.

**Features**
 • Interactive dashboard with charts, slicers, and pivot tables
 • Analysis of total orders, revenue, and delivery performance
 • Insights on top restaurants and popular food categories
 • Visualization of order trends by city, time, and category
 • Clean and user-friendly layout for easy data interpretation

**Tools Used**
 • Microsoft Excel
 • Pivot Tables
 • Charts & Graphs
 • Slicers for filtering data

**Purpose**
The goal of this project is to demonstrate data analysis and dashboard creation skills using Excel, transforming raw Swiggy dataset into meaningful visual insights.
